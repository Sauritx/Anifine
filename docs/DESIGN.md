# Design System

ANIFINE uses a premium dark gaming interface inspired by modern competitive dashboards.

## Palette

- Background: `#090909`
- Primary: `#E53935`
- Accent: `#FFC107`
- Cards: `rgba(255,255,255,0.06)`
- Typography: white and muted grey

## Typography

- Headings: Sora
- Body: Inter

## Components

Reusable primitives include GlassCard, Button, Input, Navbar, Sidebar, Modal, Dialog, Badge, Avatar, ProgressBar, XPBar, StatCard, LoadingScreen, PageTransition, and ErrorBoundary.
